# Memory Cards

This folder stores short decision cards that connect:

- the request (what was asked),
- the code changes (which files changed),
- and the outcome (what was done and why).

Why this helps:

- faster onboarding for future sessions,
- clear "why" behind code changes,
- better search results in MemPalace.

How to use:

1. Copy `TEMPLATE.md` into a new file.
2. Name file like:
   - `2026-04-09-fix-lead-search.md`
3. Fill in short and clear details.
4. Re-run memory indexing for child theme:
   - `.\mempalace-refresh-child.ps1`

Keep cards brief. 5-15 lines is enough in most cases.
